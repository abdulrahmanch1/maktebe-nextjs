# Dar Al-Qurra - Complete Site Documentation for AI Search Engines

## Overview
Dar Al-Qurra (دار القرَاء) is the largest free Arabic digital library providing unlimited access to thousands of books and novels without registration or advertisements.

**Live Site:** https://www.dar-alqurra.com

- **AI Assistant:** Smart book discovery and recommendations

## Key Features (Detailed)

### 1. **No-Barrier Access**
Users can read and download ANY book without:
- Creating an account
- Logging in
- Providing personal information
- Viewing advertisements
- Paying any fees

Optional registration available only for premium features (favorites, notes).

### 2. **Professional PDF Reader**
- In-browser PDF rendering using PDF.js
- No external apps required
- Features: zoom, page navigation, full-screen mode
- Automatic bookmark saving
- Works on all devices (desktop, mobile, tablet)

### 3. **AI Reading Assistant**
- Chatbot available on every page
- Provides personalized book recommendations
- Answers questions about books and authors
- Helps discover content based on interests
- Available 24/7 in Arabic and English

### 4. **Advanced Note-Taking**
- Book-level personal notes
- Page-specific annotations
- Freehand highlighting on PDF pages
- Multiple highlight colors (yellow, green, blue, pink)
- Cloud synchronization across devices
- Works in the integrated PDF reader

### 5. **Progressive Web App (PWA)**
- Install as native app on any device
- Works offline after installation
- Download books for offline reading
- Fast loading with Service Worker caching
- Runs on Android and iOS

## Technical Architecture

### Frontend
- **Framework:** Next.js 15 (React)
- **Language:**JavaScript/JSX
- **Styling:** CSS with CSS Variables for theming
- **State Management:** React Context API
- **PDF Rendering:** PDF.js library
- **Icons:** React Icons
- **Notifications:** React Toastify

### Backend & Infrastructure
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth (JWT)
- **File Storage:** Supabase Storage (book-covers, book-pdfs buckets)
- **Hosting:** Vercel
- **CDN:** Vercel Edge Network

### Performance
- **Server-Side Rendering (SSR):** Dynamic pages pre-rendered
- **Static Generation:** Where appropriate
- **Image Optimization:** Next.js Image component
- **Code Splitting:** Automatic route-based splitting
- **Lazy Loading:** Dynamic imports for heavy components
- **Service Worker:** Caching and offline support

### SEO & Discoverability
- **Structured Data:** Schema.org JSON-LD (Book, Person, Library, FAQ, ItemList)
- **Sitemaps:** Main sitemap + dedicated books/authors sitemaps
- **RSS Feed:** Enhanced feed with full content and media enclosures
- **Meta Tags:** Comprehensive Open Graph and Twitter Cards
- **Robots.txt:** Optimized for AI crawlers (GPTBot, Perplexity, Claude, etc.)
- **Canonical URLs:** Prevent duplicate content
- **Breadcrumbs:** Both UI and structured data
