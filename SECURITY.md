# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Dar Al-Qurra, please report it to us as soon as possible.

**Email:** security@dar-alqurra.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggestions for fixing

We take all security reports seriously and will respond within 48 hours.

## Security Measures

### Data Protection
- All user data encrypted in transit (HTTPS)
- Secure authentication via Supabase
- Minimal data collection
- No data selling or sharing

### Content Security
- Content moderation
- User-generated content monitoring
- Spam prevention
- Abuse reporting system

### Infrastructure
- Hosted on Vercel (enterprise-grade security)
- Database on Supabase (SOC 2 Type II certified)
- Regular security updates
- Automated vulnerability scanning

## Privacy
We are committed to user privacy. See our privacy practices:
- No tracking cookies
- No third-party analytics
- Optional account creation
- Transparent data usage

## Best Practices
Users should:
- Use strong passwords
- Keep browsers updated
- Report suspicious activity
- Protect their credentials

Thank you for helping keep Dar Al-Qurra secure!
